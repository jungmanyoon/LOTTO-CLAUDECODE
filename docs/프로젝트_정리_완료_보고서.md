### nul 파일
- Windows의 특수 파일로 일반적인 방법으로 삭제 불가
- 수동으로 삭제 필요:
  ```cmd
  del \\.\D:\VisualStudio\04.로또_신버전\250727_CLAUDE CODE_R0\nul
  ```
