3. 폴더주소는 상대주소로 모두 지정 할 것.

6. 수정코드를 알려줄때는 어느부분이 수정이 되었는지 수정 부위는 항상 주석문을 추가해줘
7. 코드들 수정할때 기존 코드에서 수정이 가능하면 새로운 매서드를 만들지 말고 기존 메서드에서 수정을 해줘

9. 코드와 로그 파일을 단계단계 검토해서 문제점을 확인 후 수정 코드를 알려줘

로또 1등 당첨 확률 극대화를 위한 상세 알고리즘

1. 서론

로또 1등 당첨 확률을 극대화하기 위해서는 단순히 무작위로 번호를 선택하는 것보다 역사적인 데이터와 패턴을 분석하고, 불가능하거나 매우 희박한 조합을 제외하여 효율적으로 번호를 선택하는 전략이 필요합니다. 이를 위해 아래와 같이 알고리즘을 설계합니다.

2. 번호 예측 방식 비교

2.1 각 번호를 독립적으로 예측하는 방식

장점
모델의 복잡도가 낮아져 계산 효율이 높습니다.
각 번호의 출현 빈도 등을 기반으로 예측하기 쉽습니다.
단점
로또 번호는 무작위 추출이지만, 번호 간 상호 연관성이 있습니다(중복 불가).
번호 간의 조합 패턴을 반영하지 못해 전체 조합의 특성을 놓칠 수 있습니다.

2.2 6개 번호 조합을 하나의 단위로 처리하는 방식

장점
번호 간의 관계와 조합 패턴을 모두 고려할 수 있습니다.
불가능하거나 매우 희박한 조합을 효과적으로 제외할 수 있습니다.
단점
전체 조합 수가 8,145,060개로 매우 커서 계산 자원이 많이 필요합니다.
모델의 복잡도가 높아집니다.
2.3 결론(이방법 채택)

로또 1등 당첨 확률을 극대화하기 위해서는 6개 번호의 조합을 하나의 단위로 처리하는 것이 더 효과적입니다. 이는 번호 간의 관계와 패턴을 모두 고려할 수 있어 불필요한 조합을 효율적으로 제외하고, 당첨 확률이 높은 조합을 선택하는 데 도움이 됩니다.

3. 제외 기준 설정

로또 역사 데이터(1회차 ~ 마지막회차)를 분석하여 불가능하거나 매우 희박한 조합을 제외합니다.

3.1 필수 제외 기준

이전에 나온 동일한 조합

동일한 6개 번호 조합은 한 번도 나온 적이 없습니다.
따라서 과거에 나온 조합은 제외합니다.
5개 이상 동일한 번호를 포함하는 조합

5개 이상 동일한 번호 조합은 나온 적이 없습니다.
이전 회차들과 5개 이상 번호가 동일한 조합을 제외합니다.
모두 짝수 또는 모두 홀수인 조합

모두 짝수 또는 모두 홀수인 조합은 나온 적이 없습니다.
6개 모두 짝수이거나 홀수인 조합을 제외합니다.
연속된 6개의 숫자

예: [1, 2, 3, 4, 5, 6]
연속된 6개의 숫자는 나온 적이 없습니다.
연속 숫자가 6개인 조합을 제외합니다.
일정한 간격으로 증가하는 조합

2씩 증가: [1, 3, 5, 7, 9, 11]
3씩 증가: [2, 5, 8, 11, 14, 17]
이러한 패턴의 조합은 매우 희박하거나 나온 적이 없습니다.
일정한 간격으로 증가하는 조합을 제외합니다.
6개 중 4개 이상이 연속된 숫자인 조합

예: [10, 11, 12, 13, 25, 30]
연속된 숫자가 4개 이상인 조합은 매우 희박합니다.
이러한 조합을 제외합니다.
끝자리가 모두 같은 숫자

예: [3, 13, 23, 33, 43, 6]
끝자리가 동일한 숫자가 5개 이상인 조합을 제외합니다.
번호 합계가 극단적인 값

최소 합계나 최대 합계에 근접한 조합은 희박합니다.
번호 합계가 100 미만이거나 250 이상인 조합을 제외합니다.
이전에 특정 회차 이내에 나온 조합과 4개 이상 동일한 번호를 포함하는 조합

최근 10회차 내에 나온 조합과 4개 이상 동일한 번호를 가진 조합은 제외합니다.
이는 동적 변수로 설정하여 데이터에 따라 조정합니다.
3.2 동적 제외 기준 설정 방법

과거 데이터 분석을 통해 각 기준의 최대치나 최소치를 계산하고, 이를 기준으로 제외 조건을 설정합니다.
예를 들어, 과거에 연속된 숫자가 최대 3개까지 나왔다면, 연속 숫자 4개 이상의 조합을 제외합니다.
이러한 기준은 새로운 데이터가 추가될 때마다 업데이트됩니다.

4.1 데이터 수집 및 전처리

데이터 수집

Python의 requests 라이브러리를 사용하여 공식 로또 사이트에서 회차별 당첨 번호를 크롤링합니다.

데이터 전처리

패턴 분석

연속된 숫자 출현 패턴, 홀짝 비율, 번호 합계 분포 등을 분석합니다.
matplotlib나 seaborn을 사용하여 시각화합니다.

규칙 관리 클래스 구현

DynamicRules 클래스를 생성하여 규칙을 관리합니다.

동적 변수 설정

과거 데이터를 기반으로 최대값, 최소값 등을 계산하여 변수로 설정합니다.
예: 최대 연속 숫자 개수, 최대 동일 끝자리 숫자 개수 등.

조합 생성

조합 필터링

생성된 조합에 대해 필수 제외 기준을 적용합니다.
멀티프로세싱(multiprocessing)을 사용하여 필터링 속도를 향상시킵니다.
메모리 효율을 위해 제너레이터나 이터레이터를 사용합니다.
조합 수 감소 효과

필수 제외 기준을 적용하면 전체 조합 수는 크게 감소합니다.
예를 들어, 제외 기준을 적용하여 조합 수를 1,000,000개 이하로 줄일 수 있습니다.

데이터 준비

랜덤포레스트(RandomForestClassifier), 그래디언트 부스팅(XGBoost), 심층신경망(Deep Neural Network) 등을 사용합니다.
