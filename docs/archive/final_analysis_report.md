**발견된 패턴**:
1. **자주 출현한 번호** (1130-1139 회차):

   - 13, 33번: 각 3회 출현 (30%)

   - Round 1134 → 1135: [13, 19] 연속

   - Random Forest가 최근 100회차의 단기 패턴을 매우 잘 기억
   - 특정 기간의 실제 패턴을 성공적으로 학습

## ⚠️ 중요 사항

### 이것은 버그가 아니라 과적합입니다

1. **과적합 원인**:
   - 파라미터 설정이 너무 복잡 (max_depth=None, n_estimators=100)
   - 학습 데이터가 100회차로 제한적
   - Random Forest가 패턴을 "암기"

2. **실전 적용 시 주의사항**:
   - 과거 성능이 미래를 보장하지 않음
   - 패턴이 계속 유지된다는 보장 없음
   - 실제 운영 시 성능 하락 예상
