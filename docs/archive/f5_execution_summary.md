## 🚀 실행 순서 (F5 키)

```
1. 데이터 수집 → 2. 패턴 분석 → 3. ML/AI 예측 → 4. 백테스팅
→ 5. 필터링 → 6. 예측 생성 → 7. 모니터링 → 8. 피드백 루프
```

## ⏱️ 예상 실행 시간

| 단계 | 기존 | 최적화 후 |
|------|------|-----------|
| ML/AI 예측 | 60초 | 30-40초 |
| 백테스팅 | 200초 | 60-80초 |
| 전체 실행 | 5-6분 | 2-3분 |

## 📁 생성되는 파일

1. **예측 결과**: `results/lotto_predictions_[날짜].json`
2. **백테스팅**: `results/backtest_results_optimized_[날짜].json`
3. **성능 보고서**: `results/performance_report_[날짜].txt`
4. **모델 캐시**: `cache/models/[모델명]_[해시].pkl`

## 🔍 확인 방법
