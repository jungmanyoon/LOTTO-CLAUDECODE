## 🛠️ 필터 조정 메커니즘

### 1. 정적 필터 → 동적 필터 전환

```yaml
# 기존 (config.yaml)
filters:
  sum_range:
    min_sum: 100  # 고정값
    max_sum: 170  # 고정값

# 자동 조정 후 (config_dynamic.yaml)
filters:
  sum_range:
    min_sum: 95   # 최근 패턴 기반 자동 조정
    max_sum: 175  # 최근 패턴 기반 자동 조정
  
  hot_cold:  # 새로 추가된 동적 필터
    hot_numbers: [3, 7, 15, 23, 31, 39]
    cold_numbers: [2, 11, 18, 26, 34, 41]
    min_hot_numbers: 2
    max_cold_numbers: 3
```

### 2. 조정 알고리즘

```python
조정 기준값 = f(최근_패턴, 백분위수, 변화율)

where:
  - 최근_패턴 = analyze_recent_patterns(window=100)
  - 백분위수 = percentile([10, 90])  # 극단값 제외
  - 변화율 = (현재값 - 과거평균) / 과거평균
```
